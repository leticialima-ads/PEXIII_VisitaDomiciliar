# 🏥 PEX III – Sistema de Visita Domiciliar

## 📌 Sobre o Projeto

O **PEX III – Sistema de Visita Domiciliar** é um aplicativo desenvolvido com o objetivo de auxiliar profissionais da saúde no registro, organização e acompanhamento de visitas domiciliares.

A aplicação foi pensada para tornar o atendimento mais ágil, seguro e eficiente, permitindo acesso rápido às informações dos pacientes e histórico de visitas.

---

## 🎯 Objetivo

Facilitar a gestão de atendimentos domiciliares, promovendo:

* Organização dos dados dos pacientes
* Registro estruturado das visitas
* Acompanhamento contínuo do histórico clínico
* Apoio na tomada de decisão durante o atendimento

---

## 🚀 Funcionalidades

* 📋 Cadastro de pacientes
* 🔍 Pesquisa de pacientes por nome
* 🏠 Registro de visitas domiciliares
* 📊 Visualização de relatórios
* 📁 Histórico de atendimentos

---

## 🛠️ Tecnologias Utilizadas

* Flutter
* Dart
* (Backend em desenvolvimento)

---

## 📱 Interface

Interface simples e intuitiva, pensada para uso rápido durante visitas em campo, facilitando o acesso às informações mesmo em situações de tempo limitado.

---

## ▶️ Como Executar o Projeto

```bash
# Clone o repositório
git clone https://github.com/leticialima-ads/PEXIII_VisitaDomiciliar.git

# Acesse a pasta do projeto
cd PEXIII_VisitaDomiciliar

# Instale as dependências
flutter pub get

# Execute o projeto
flutter run
```
